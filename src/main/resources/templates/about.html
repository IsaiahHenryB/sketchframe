About

        
        
SketchFrame is an app that allows users to generate abstract art using a JavaScript library called p5.js and
            upload it to their profiles. People on the site can view all the public creations of the previous users and
            even download them.


        
So What's Up With This Website?

        
This is a website that utilizes just a few properties of a JavaScript framework called p5 to generate art.
            The art generated is based on strokes and outlines. The JavaScript code randomizes the starting angle of the
            first stroke and color palette, then creates proceeding strokes next to the first generated stroke depending
            on the angle selected. Tweaking parameters can yield nearly endless possibilities.


        
On the create page you can set your own parameters with the controls and see this in
            action for yourself.

        
Inspiration

        
The art creator is based off of a p5 demonstration by Steve's Makerspace. Please check
            out his youtube channel for more p5 goodies! Though this abstract art generator does quite a bit of things,
            you will find that there's literally endless potential for what you can do using the great JavaScript
            library called p5.js!
