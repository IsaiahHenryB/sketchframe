Welcome To SketchFrame 

        
If you're interested in AI and art, this is the place for you!
SketchFrame is a platform where you can use AI to generate abstract art from simple shapes. There are virtually endless possibilities of generated you can create because every output is different from the last!

        
You Can Start Now!

        
If you want to start making cool abstract art, simply navigate to the create page!
There you can create custom art by tweaking the parameter values that are used to generate abstract art. You will be surprised at the variety of outcomes you can achieve!


        
View The Art That Other People Created

        
Are you curious what other people have created on this website? Feel free to check out all the artworks that have been uploaded by users on the Community Gallery page. Don't feel left out! If you create an account, you can add pieces of your own the community gallery!


            
Let Your Imagination Run Wild... But Not TOO Wild

            
*DISCLAIMER*

            
While creating art on this app, there are different values that you can input into the art generator. Depending on the number of strokes the generator has to create, there can be a great variation in load times. This all may vary based on your hardware and/or browser. Be wary that generating too many strokes can even lead to crashing your browser if the memory usage is too high.
